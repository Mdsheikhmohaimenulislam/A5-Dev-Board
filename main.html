Question-1: What are the different ways to select an element in the DOM?

getElementById শুধু একটা element এর জন্য কাজ করে,, getElementsByClassName উপাদান গুলোর জন্য কাজ করে,, getElementsByName নামের উপাদানের উপর কাজ করে এবং ফর্ম উপাদানগুলোর জন্য ব্যবহিত হয়,, querySelector এ css নির্বাচকের উপাদানের জন্য কাজ করে,,, querySelectorAll css এর অনেক গুলো Class নাম একসাথে মেলে কাজ করে।

Question-2: What is the difference between innerHTML, innerText, and textContent ?

innerHTML এর মধ্যে আমার dynamic HTML tag ব্যবহার করা যায়,,, innerText এর মধ্যে আমরা read the content of element and return the text and html tag ignores করে and style or modifying the innerText,,, textContent সব Html tag কে ignores করে এবং text return করে। আর textcontent এর মধ্যে style add করা যায় না

Question-3: What is event delegation in the DOM?

একটি একক eventList কে একটি parent element যোগ করা হয়, এবং একাধিক child element আলাদা ইভেন্টগুলি যোগ করা হয় এবং তারপর event tag ব্যবহার করে elements delegated বের করা হয়।

Question-4: What is event bubbling in the DOM?

যখন কোন এলিমেন্টে ইভেট্ন ঘটে তখন এটি প্রথমে তার ইভেন্ট handler চালায়,এবং তার object না পাওয়া পর্যন্ত bubbles চালায়।
